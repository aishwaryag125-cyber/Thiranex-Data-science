# Data Cleaning & Visualization Project

## Dataset
Student Performance Dataset from Kaggle

## Objectives
- Handle missing values
- Detect duplicates
- Detect outliers
- Visualize important trends

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
1. GPA Distribution
2. Gender Distribution
3. Study Time vs GPA
4. Absences vs GPA
5. Correlation Heatmap

## Key Findings
- More study time generally improves GPA.
- More absences are associated with lower GPA.
- Dataset contains minimal data quality issues.
